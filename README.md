## Clone and Install Package

### Clone repository

```js
git clone https://github.com/vophuocthanh/SE397_Travel.git
```

### Install package

```js
yarn install
```

### Run project

```js
yarn run dev
```

## Tech stack

- React + TypeScript + Vite
- TailwindCSS + Shacdn UI
- React Query + TanStack Query
